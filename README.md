## Setup

1. Get the code:

  ```
  git clone https://github.com/tysongach/taskafe.com.git
  ```

1. Install dependencies:

  ```
  bundle install
  ```

1. Verify that the app is up and running

  ```
  bundle exec middleman
  ```

  ```
  open http://localhost:4567
  ```
